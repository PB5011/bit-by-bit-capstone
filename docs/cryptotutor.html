callNicad(threshold)¶

Function for running the NiCad process.

This function calls the nicad6cross process to operate on the code that is submitted by the student. The code must already exist within the submissions folder, which is defined by the location variable.

Parameters

  • studentUsername – The student’s username

  • threshold – The threshold that NiCad should run at. This is an alias for the configuration filename located within the NiCad installation directory.

cleanNicad()¶

Function for cleaning the student’s NiCad folder.

This function cleans the NiCad folders associated with the student’s username. Used primarily for when NiCad encounters errors, as it was discovered NiCad output should be cleaned upon error.

Parameters

studentUsername – The student’s username.

cryptotutor.views.error(request, exception, log, additionalMessage)¶

View function for exception handling. This view allows for developers to render a page containing error details from a caught exception. This method takes in exceptions, log files, and any additional messages to render on the page for the user to see.

Parameters

  • request – A request to view the exception page.

  • exception – A Python exception object containing the thrown exception.

  • log – A path to a log file on the filesystem.

  • additionalMessage – A string containing any additional messages to pass to the user.

Return type

HttpResponse

Returns

An HttpResponse with the original request, the change password page url, and the context of the page.

cryptotutor.views.index(request, sort_type='')¶

View function for home page of site.

Retrieves all questions from the database. By default, sorts by newest questions first, however a different sort_type may be used to sort the results accordingly.

Parameters

  • request – A request to view the home page.

  • sort_type – The type of sorting to be applied to the questions. Defaults to newest questions first; options include popularity determined by points, popularity determined by views, time (newest and oldest first), and number of answers.

Return type

HttpResponse

Returns

An HttpResponse with the original request, the home page url, and the context of the page. Context of the page includes all questions from the database.
